This is the admin dashboard. It shows total users of the system :

Total Alumni users, total number of administrators and total extension workers will be displayed here. Total system users information and proportion of users in the system also will be displayed here.

Description For Admin Dashboard:

 

 

 

System Users Information

 

 

The system users are Admin,extension worker and Users. All of this system users system will be displayed here within a table. It is possible to sort the information by pressing the arrow beside table headers.

 

 

 

 

 

 

Proportion Of Users In The System

 

 

It shows the proportion each user has in the system and it is represented using pie chart.

 

 

 

 

 

 

 

Total Users (Partners)

 

 

It shows total number of alumni users in the system such as FAO, ATA,MOA..

 

 

 

 

 

 

Total Number Of Admins

 

 

It shows total number of system admins that are controlling the system.

 

 

 

 

 

Total Extension Workers

 

 

It shows total number of extension workers in the system.

 

 

 

 

 

 

 

Total Seed Experts

 

 

It shows total number seed experts in the system.
