Farmers do not have direct access to the system. Only the extension worker does the work of registering farmers and managing farmers. Farmers subscribe for the extension workers service after registration.

 

•What this page mainly does is managing farmers ( adding farmer, editing farmers information and deleting farmer subscribed to the extension worker).

•It is also possible to sort the table record by pressing the up arrow beside the table header.

 

 

                                Farmers Page

 

 

 

 

 

 

 

Description For  Farmers Page:

 

 

 

Add New Farmer Button

 

 

It is add new Farmer button. When it is clicked it displays new page called Add New Farmer.

 

 

 

 

Farmers Record Table

 

 

It is Farmers information table. It contains farmers name, city and other records. It is possible to perform edit and delete operations from here.

 

 

 

Edit Farmer Record

 

 

Edit option enables the user to modify previously inserted record. By pressing the edit button it is possible to edit the specified farmer record under the extension worker subscription.

 

 

 

Delete Farmer

 

 

Delete option enables the user to delete previously recorded farmer data. By pressing the delete button it is possible to delete the specified farmer record under the extension worker subscription.

 

 

 

 

 

 

 

                      Record Farmers Information

 

 

 

 

 

 

Description For  Record Farmers Information:

 

 

 

 

Farmer Name

 

 

 

Insert the farmer name here.

 

 

 

 

City

 

 

Insert the city where the farmer is located here.

 

 

 

Farmer Phone Number

 

 

Insert the farmer phone number here.

 

 

 

Submit Button

 

 

Finally press the submit button after all the above forms are fully filled.
