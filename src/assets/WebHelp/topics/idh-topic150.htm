In this page the extension worker can manage harvest information: adding new harvest record, editing previously inserted harvest records and deleting harvest records of a farmer).

It is also possible to sort the table record by pressing the up arrow beside the table header.

 

                                 Harvest Record Page

 

 

 

 

 

Description For  Harvest Record:

 

 

 

Add New Harvest

 

 

It is add new harvest record button. When it is clicked it displays new page called New Harvest.

 

 

 

 

 

Harvest Information Table

 

 

It is Harvest  record table. It contains farmers name, crop type, quantity , quality and other records. It is possible to perform edit and delete operations from here.

 

 

 

Edit Harvest Information

 

 

Edit option enables the user to modify previously inserted record. By pressing the edit button it is possible to edit the specified farmer harvest record under the extension worker subscription.

 

 

 

Delete Harvest Data

 

 

Delete option enables the user to delete previously  recorded farmer harvest record. By pressing the delete button it is possible to delete the specified farmer harvest record under the extension worker subscription.
