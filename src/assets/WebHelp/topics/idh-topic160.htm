In this page the extension worker can manage plant record: adding new plant record, editing previously inserted plant records and deleting plant records of a farmer).

It is also possible to sort the table record by pressing the up arrow beside the table header.

 

                                     Plant Record Page

 

 

 

 

 

Description For  Plant Record Page:

 

 

 

Add New Plantation

 

 

It is add new plantation record button. When it is clicked it displays new page called New Plantation.

 

 

 

 

 

Plantation Record Table

 

 

It is plantation record table. It contains farmers name, crop type,  year of plantation, variety, seed-rate, quantity ,

quality and other records. It is possible to perform edit and delete operations from here.

 

 

 

Edit Plantation Record

 

 

Edit option enables the user to modify previously inserted record. By pressing the edit button it is possible to edit the specified farmer plantation record under the extension worker subscription.

 

 

 

Delete Plantation Record

 

 

Delete option enables the user to delete previously recorded farmer plant record. By pressing the delete button it is possible to delete the specified farmer plant record under the extension worker subscription.
