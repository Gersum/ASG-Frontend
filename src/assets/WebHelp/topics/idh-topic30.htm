This is the homepage the admin will be seeing when he/she logs into the system as an admin. The privilege of the administrator is managing the user accounts in the system from extension expert to the farmers. The farmers will not have an account themselves but since the extension expert record the data of each famer and  the admin access and control the information.

Some of the things an admin does:

•Creating account for extension experts and other admins.  

•Managing system users like passwords, usernames and create and delete accounts.
