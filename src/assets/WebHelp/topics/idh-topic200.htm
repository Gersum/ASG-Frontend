This is the page that a new user to the system will be added. The admin has the responsibility to add a new user to the system and assigning a role to the users like extension worker role.

 

 

                                   Create Accounts

 

 

 

 

Description For Create Accounts:

 

 

 

User Name

 

 

Insert the user name here.

 

 

 

 

 

 

Email

 

 

Insert user email address here.

 

 

 

 

 

 

Password

 

 

Create a new password to the user here.

 

 

 

 

 

 

Role

Insert the role of the user like admin, extension worker etc ..

 

 

 

 

 

 

Create New Account Button

 

 

Finally press the create new account button after all the above forms are fully and correctly filled. Then the system creates new user with a given role.
