This page includes creating, editing and deleting accounts from the system by the admin. The admin has the responsibility to creat, edit and delete user accounts from the system.

 

                                        Accounts 

 

 

 

 

Description For Accounts Page:

 

 

Add New User

 

 

It is add new user button. When it is clicked it displays new page called Add New User.

 

 

 

 

 

 

Users Information

 

 

It is users information table. It contains users name, email, role and date created. It is possible to perform edit and delete operations from here.

 

 

 

 

 

Edit and Delete Options

 

 

Edit option enables the admin to modify previously inserted record. By pressing the edit button it is possible to edit the specified user record.

Delete option enables the admin to delete previously recorded user data. By pressing the delete button it is possible to delete the specified user record..
